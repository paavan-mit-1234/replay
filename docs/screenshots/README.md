# Screenshots

Drop PNGs here with these exact names and they'll appear in the main README:

| File | What to capture |
|------|-----------------|
| `hero.png` | The landing page hero (logged-out home), or a striking shot of the chat/dashboard |
| `chat.png` | The chat workspace — a conversation with the sidebar, "improve my prompt", saved prompts |
| `dashboard.png` | The observability dashboard — gauges, budget bar, trend charts, live request stream |
| `evals.png` | The eval harness — the runs table showing a regression (v1 100% vs v2 0%, expanded results) |
| `insights.png` | The insights tab — stats gauges + a generated prompt fingerprint |
| `playground.png` | The in-app playground (or a request detail view) |

Tip: capture at a wide viewport (~1400px) for crisp, full-width images. The app's
"analog signal laboratory" look (hard edges, flat color, mono numerals) photographs well.
